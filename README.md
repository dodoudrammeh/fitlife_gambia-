# 🏃‍♂️ FitLife_Gambia

This is my **first project** built when I joined **Jasseh Code Camp (JCC)**. The project was created following a roadmap designed by the founder of JCC, **Omar**, to guide me through my first steps in web development.

**FitLife_Gambia** helped me practice building structured pages using **HTML** and styling them with **CSS**, giving me hands-on experience as a beginner.

## 🚀 Features

* Simple and clean layout
* Built entirely with HTML and CSS
* Introductory project to practice frontend fundamentals

## 📚 What I Learned

* Writing organized HTML
* Styling with pure CSS
* Following a development roadmap
* Taking guidance from an experienced mentor
